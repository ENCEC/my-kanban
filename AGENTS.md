# AGENTS

## Project Rules

- All documents generated through speckit in this repository must use Chinese by default.
- Proper technical nouns, commands, paths, protocol names, library names, model names, file names, and other established technical identifiers must remain in their original form and should not be translated forcibly.
- Requirement keywords, acceptance-template words, and normative words such as `must`, `Given`, `When`, and `Then` should be translated into natural Chinese in generated documents.
- If a user explicitly requests a one-off exception for a document, follow the user request for that document only.

## Source Of Truth

- `.specify/memory/constitution.md` is the project-level authority for speckit document conventions.
